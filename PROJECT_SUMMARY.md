## 🎨 Design System

### Color Scheme
- **Primary (Green):** `#10b981` - Trust, growth, action
- **Secondary (Orange):** `#f97316` - Energy, enthusiasm, urgency
- **Accent (Dark Blue):** `#1e3a8a` - Professional, reliable, corporate

### Typography
- **Sans-serif fonts** (system default + Google Fonts ready)
- **Large readable headings** (Section titles 3xl-4xl)
- **Clear hierarchy** (H1 → H2 → H3)

### Layout
- **Container max-width:** 1280px
- **Mobile-first responsive** design
- **Spacing:** Consistent padding/margins
- **12-column grid system** (Tailwind built-in)

### Components & Patterns
- **Buttons:** Primary (green), Secondary (orange)
- **Cards:** Shadow, hover effects, transitions
- **Sections:** Alternating backgrounds (white/gray)
- **CTAs:** Multiple per page, high contrast, clear actions
